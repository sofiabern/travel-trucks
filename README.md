# TravelTruck Frontend

This project is the frontend for the TravelTrucks application. It is built using React to provide an intuitive interface for searching, filtering, and managing truck rentals tailored for travel enthusiasts.

## Demo
You can access the live demo of the application here: [TravelTrucks Frontend](https://travel-trucks-nu-two.vercel.app/)

## Features
- **User-Friendly UI:** Clean and intuitive interface to ensure a smooth experience.
- **Detailed Truck Profiles:** View comprehensive information about each truck, including images, features, and rental prices.
- **Truck Search:** Easily search for available travel trucks based on location, equipment, and type.
- **Favorites Management:** Save and manage your favorite trucks for future reference.

## Technologies
This project is built with the following technologies:
- **React:** For building a dynamic and responsive UI.
- **Redux:** For managing application state.
- **React Router:** For client-side navigation.
- **CSS Modules and Styled Components:** For a clean and component-driven UI.
- **Axios:** For API interactions.
- **Vite:** For fast development and build processes.

## Getting Started

To run the project locally, follow these steps:

1. **Clone the repository**
    ```sh
    git clone https://github.com/sofiabern/travel-trucks.git
    ```

2. **Navigate to the project directory**
    ```sh
    cd travel-trucks
    ```

3. **Install dependencies**
    ```sh
    npm install
    ```

4. **Run the application**
    ```sh
    npm start
    ```

   The application will be available at `http://localhost:5173` by default.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `build/` directory.
